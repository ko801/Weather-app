A real-time weather application built with React.js
that fetches live weather data using the OpenWeatherMap API.

## 🔗 Live Demo
[Click here](paste-your-live-link-here)

## ✨ Features
- Search weather by city name
- Shows temperature, humidity and weather condition
- Real-time data from OpenWeatherMap API
- Clean and responsive UI

## 🛠️ Tech Used
- React.js
- JavaScript
- OpenWeatherMap API
- CSS

## 🚀 How to Run
1. Clone the repo
2. Run npm install
3. Add your API key in the code
4. Run npm start
